Kalkulator Pajak OOP

### Tentang  
Aplikasi ini sebagai latihan saya belajar python.  
  
  
### Skenario
* Bujang / belum menikah:
  * Lembur perjam 8000
  * Pajak 10% dari total gaji
  * Asuransi 100000
  * Jika gaji diatas 3000000:
    * Pajak 15%
    * Asuransi 120000
 * Menikah:
   * Lembur perjam 9000
   * Pajak 10% dari total gaji
   * Asuransi 120000
   * Jika gaji diatas 3000000:
     * Pajak 15%
     * Asuransi 150000
   * Tunjangan 40000
   * Tunjangan per anak 30000
  
  
### Instalasi
```console
$ pip install -r requirements.txt
$ ./main.py
```
